  • Doctype作用？严格模式与混杂模式如何区分？它们有何意义?

    • doctype启用标准模式 根据css规范渲染页面，否则则以兼容模式渲染，兼容模式会模拟老式浏览器进行渲染 会造成 怪异的css渲染

  • 行内元素有哪些？块级元素有哪些？ 空(void)元素有那些？

    • 行内元素: span a b i em strong
    • 块级元素: div h1 p section article ul li ol
    • 空元素: img hr br link meta

  • 页面导入样式时，使用link和@import有什么区别？

    • link: 会阻塞页面渲染，等link指向的样式文件被解析后才会渲染下面的元素
    • @import: 等页面渲染完后加载文件

  • 介绍一下你对浏览器内核的理解？

    • js引擎主要用于执行脚本文件

    • 渲染引擎主要用于解析dom和css规则,并进行排版到浏览器 显示现有的样式界面

      解析 HTML 构建 DOM 树 -> 构建渲染树 -> 渲染树布局 -> 渲染树绘制。

  • html5有哪些新特性？如何处理HTML5新标签的浏览器兼容问题？如何区分 HTML 和 HTML5？

    • 新特性:

      • canvas 绘画

      • localstorage sessionstorage 离线储存

    • 处理浏览器兼容

      • 可以使用document.createElement(tag)来创建元素

      • 可以在页头加上 判断 如果是IE9以下的浏览器 加上垫片

    • 如何区分HTML5

      • DOCTYPE声明\新增的结构元素\功能元素

  • 简述一下你对HTML语义化的理解？

    • 利于代码的阅读
    • 利于SEO，利于爬虫的解析

  • 请描述一下 cookies，sessionStorage 和 localStorage 的区别？

    • cookies
      • 是为了标识用户身份而存在用户端的数据
      • cookies的所有数据始终在同源http请求中携带，即使不需要
    • sessionStorage
      • 数据当浏览器关闭后自动删除
    • localStorage
      • 除非主动删除数据，否则不会丢失
    • cookies储存数据为4k,localStorage和sessionStorage的数据最多可达到5M以上

  • iframe有那些缺点？

    • 会阻塞主页面onLoad事件，该事件会等待页面所有元素解析完后执行
    • 搜索引擎无法检索这种页面，不利于seo
    • iframe和主页面共享连接池，而浏览器对相同域的连接有限制，所以会影响页面的并行加载。

  • title与h1的区别、b与strong的区别、i与em的区别？

    title没有意义仅为网页标题，h1则表示层次明确的标题，对SEO有很大影响

    strong表示强调语气，在语音阅读网页时会重读，b标签为强调内容

    i表示斜体，em表示强调文本

  • 介绍一下标准的CSS的盒子模型？与低版本IE的盒子模型有什么不同的？

    3. 区别: IE盒子模型会把padding和border算到content中怪异模式下出现

  • 如何居中div？如何居中一个浮动元素？如何让绝对定位的div居中？

    • 水平居中div

      • margin:0 auto;(需要div有宽度，因为默认是100%)

    • 浮动元素无法居中，所以需要给浮动元素增加父元素 使父元素居中

    1. 相对定位垂直居中 缺点:该元素占据常规文档流空间，但是显示在偏移位置，会导致文档流很诡异

    2. 绝对定位垂直居中

    3. flex 兼容性差

  • display有哪些值？说明他们的作用。

    • inline:内联，无法设置高宽度
    • block:块，默认宽度为百分百，高度随内容撑开。
    • inline-block:内联块，具有内联的特征，可以被text-align:center居中，默认高宽跟随内容
    • none:元素不显示，并从文档流中移除

  • position的值relative和absolute定位原点是？

    • absolute 以第一个不为static的父级元素作为原点
    • relative 以任意父元素作为原点
    • fixed 以窗口作为原点
    • static 默认值 无法定位 left right等定位属性 无效

  • 请解释一下CSS3的Flexbox（弹性盒布局模型）,以及适用场景？

    采用flex-flow很方便的实现居中，采用flex布局的元素称为”容器”，每个子元素都是一个”项目”，特点:非常方便居中，可以根据屏幕大小自适应尺寸。

    适用常见为：适应h5的浏览器，常用于移动端Web页面

  • 用纯CSS创建一个三角形的原理是什么？

    均分原理，在矩形的直角，两条边的样式要均分，当一条边为透明时，另一条边均分，就形成了一个45度角

  • style标签写在body后与body前有什么区别？

  • 解释JavaScript原型，原型链 ? 有什么特点？

    • 每个对象都有一个原型，当我们访问一个对象不存在的属性时，就会顺着原型链一层层找下去

    • 对象和构造器都有原型，关系为 关系：instance.constructor.prototype == instance.__proto__

  • JavaScript有几种类型的值？，你能画一下他们的内存图吗？

    分别为 原始数据类型 和 引用数据类型

    两者区别是：存储的位置不同

    原始数据类型因为占据空间小，大小固定，属于频繁使用数据 所以放入栈内存中

    引用数据类型因为占据空间大，大小不固定，如果存在栈内存中会影响程序性能，所以存放在堆内存中，栈内存中使用引用作为凭证，当需要操作该内存时，再根据引用找到真实内存进行操作

    1. 原始数据类型：string number undefined boolean null
    2. 引用数据类型：数组，对象，函数
